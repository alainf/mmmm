== LEADS ==

* 3 types de blocs: affichage = "article" ou "breve" ou "resultat"
* dans chaque ressource, il y aura un attribut "lead" = référence vers le lead de la ressource.

=== Articles ===

- ressource					= référence vers l'objet de ce LEAD.
- pertinence				= nombre float entre 1 et 10.
- href						= url pour l'attribut HREF.
- titre						= string
- soustitre					= string
- citation					= string ou HTML ou Markdown
- contenu					= string ou HTML ou Markdown
- thumb
- thumb.beside				= "headline" ou "content"
- thumb.float				= "left" ou "right"
- thumb.tip					= string qui ira dans l'attribut "title" de la balise "img".
- thumb.url					= url qui ira dans l'attribut "src" de la balise "img"

=== Breves ===

- ressource		= référence vers l'objet de ce LEAD.
- pertinence	= nombre float entre 1 et 10.
- href
- titre
- soustitre
- contenu

=== Resultat (de recherche) ===

- ressource		= référence vers l'objet de ce LEAD.
- pertinence	= nombre float entre 1 et 10.
- href
- titre
- soustitre
- contenu
